## Preparation
Dans un dossier, vous aurez des dossiers et fichiers dans cette configuration :

```
Tp3
 |----- www.Tp3.com
             |-------nginx
                        |-------default.conf
             |-------php
                        |-------Dockerfile
             |-------php2
                        |-------Dockerfile
             |-------serveur1
                        |-------certs
                                  |------- server.crt
                                  |------- server.key
                        |-------conf
                                  |------- httpd.conf
                                  |------- httpd-ssl.conf
                                  |------- httpd-chost.conf
                        |-------html
                                  |------- index.php
                        |------- Dockerfile
             |-------serveur2
                        |-------certs
                                  |------- server.crt
                                  |------- server.key
                        |-------conf
                                  |------- httpd.conf
                                  |------- httpd-ssl.conf
                                  |------- httpd-chost.conf
                        |-------html
                                  |------- index.php
                        |------- Dockerfile             
             |-------docker-compose.yml
```
nous allons maintenant passer sur chaqu'un de fichier pour preparer la configuration.
